# BuildOps Chakra

Factory Operations ERP built on Frappe. Operator management, attendance tracking, and payments for BuildOps.

## Modules
- **FF (Factory Functions)** - Projects, Attendance, Roster management
- **Operator** - People database with assignment tracking
- **Attendance Discrepancy** - Operator-raised flags for incorrect attendance

## Roles
- **System Manager** - Full access
- **FF Chief** - Add operators, mark attendance, view projects
- **FF Operator** - View own profile, attendance, raise discrepancies
- **FF Vendor** - Invoice submission and payment tracking
